# Media Assets

This directory contains media assets used throughout the IoTflow project, such as demonstration videos, and supporting images.

All media for this project is stored under a youtube playlist linked below: 

[`MakeThingsHappy IoTflow Media Youtube`](https://www.youtube.com/playlist?list=PL9lLLGh5-GZTbbmngdXERB-36slZDMHHg)

## 📦 Purpose

The `/Media` folder provides a centralized location for non-source files that enhance documentation and help users understand the hardware and software components.

Contents include:

- Project demo videos  
- Hardware walkthrough recordings  
- Images or renders
